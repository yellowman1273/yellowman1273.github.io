FLOP

est. 1972

warning: this programming language is fictional. take this for a grain of salt.

FLOP is a general-purpose programming language made in 1972. FLOP gives a (somewhat) human-readable syntax while giving low-level access to programmers. This is why FLOP is commonly used in many sorts of stuff that needs low-level access.

Current stable version : FLOP-17

Current preview version : FLOP-23
